# Professional Tools Website

A modern multi-tool web application containing 10+ useful daily tools including:

- Age Calculator
- EMI Calculator
- BMI Calculator
- Unit Converter
- Image Compressor
- Password Generator
- Text Formatter
- URL Encoder/Decoder
- Tip Calculator
- Random Name Generator

## Features
- Clean, professional UI
- Fast and fully responsive
- Dark mode with local storage
- SEO optimized tool pages
- AdSense-friendly layout
- Privacy, Contact, Disclaimer, Sitemap included
- 100% client-side (no backend needed)
